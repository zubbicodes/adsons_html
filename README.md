# ADSONS Group of Industries

Certified preshrink elastic manufacturer leading the way in sustainable manufacturing and innovation. ADSONS specializes in polyester knitted elastics, yarns, tapes, cords, and more, trusted by top brands globally.

## About The Project

ADSONS Global is a comprehensive website showcasing the products, manufacturing processes, and sustainability initiatives of ADSONS Group of Industries. The website highlights the company's commitment to eco-friendly practices, quality certifications, and its extensive portfolio of textile products.

## Key Features

- **Product Showcase:** Detailed catalog of elastics, tapes, cords, and yarns.
- **Sustainability Focus:** Information on eco-friendly materials, energy efficiency, and waste reduction.
- **Global Certifications:** Display of industry certifications (Oeko-Tex, Higgs, etc.).
- **Events & Workflow:** Insights into company events and manufacturing workflows.
- **Responsive Design:** Optimized for all devices using Bootstrap.

## Technologies Used

- **HTML5**
- **CSS3 / SCSS**
- **Bootstrap 5**
- **JavaScript (jQuery)**
- **AOS (Animate On Scroll)**
- **Owl Carousel**

## Getting Started HERE

To view the project locally:

1.  **Clone the repository:**
    ```bash

    ```

2.  **Open `index.html`:**
    Simply open the `index.html` file in your web browser. It will redirect you to `html/index.html`.
    Alternatively, navigate directly to `html/index.html`.
